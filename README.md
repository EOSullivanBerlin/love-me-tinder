# Love me Tinder
A dating / relationship compatibility game / simulator.

## Installation
* git clone git@github.com:langphil/love-me-tinder.git
* bundle install
* bin/rails db:create
* bin/rails db:migrate
* rails s
* http://localhost/3000

## Other
* rspec (runs tests)
* rubocop (code quality)
